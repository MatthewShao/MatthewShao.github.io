發表於 2018-12-31 | 分類於 碎念

也让今日用回忆拥抱过去，用希望拥抱着未来——纪伯伦

写于2018年的最后一小时。
看的书多了反而不敢轻易表露自己的情感，就像朋友圈的配图永远都放不满九张，反反复复编辑好多次才能按下发表的按钮。害怕自己的思想过于浅显引人发笑，或者是担忧此刻的情绪遭到未来时光的审判。
于是只能把今年的书单列下来充作年终总结，却也发现能够恰到好处地勾勒出今年的轨迹。

  • 菊与刀 / Ruth Benedict 其实一直没有看完
  • 一个人到两个人：银河系亲密关系实用指南 / KnowYourself, 知乎 有理有据，然而爱情是最不讲道理的
  • 寂寞的游戏 / 袁哲生 有时也想有一艘潜水艇
  • 刺杀骑士团长 / 村上春树 在kindle上面看完的，家里供着一套还没有拆封，都看完了还没拆封也许就不会拆封了
  • 假性亲密关系 / 史秀雄 从原生家庭与自我认知了解人际交往
  • 百年法 / 山田宗树 出生和死亡算得上是人生的两项头等大事，围绕这两件事情，作者从方方面面作了探讨
  • 雪球专刊164期——如何赚到人生第一桶金 / 雪球用户, 雪球 种下一棵树最好的时间是——树苗最便宜的时候
  • 银河帝国1~3(基地、基地与帝国、第二基地)　/ Isaac Asimov 心理史学名不虚传
  • 敦煌 / 井上靖 旅途中重读了一次
  • 奇迹 / 是枝裕和 其实奇迹每天都会上演
  • 先知 / 纪伯伦 诗集
  • 如父如子 / 是枝裕和 作者真的很擅长写小朋友
  • 恋爱从失恋开始 / 林孑 唉
  • 基督山伯爵 / 大仲马 其实世界上只有一种超能力：超级有钱
