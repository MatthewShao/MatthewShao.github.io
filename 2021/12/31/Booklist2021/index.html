Booklist2021

發表於 2021-12-31

今年是充满转折的一年，而所有的变化似乎都集中到了那个闷热压迫的7月份。从校园到办公室，从广州到上海，从爱好者到从业人员。
从住了20年的房子尽力想把一件件记忆打包搬离，但那些成长过程中的酸甜苦辣再也无处寄存，只知道从此多了一个再也回不去的故乡。
但这些变故并没有在我身上产生预想中激烈的情绪变化，我能更敏锐地察觉内心那些不安的想法，然后告诉自己这些都是小事。也许是这些年的经历始终没有白走，但我更愿意相信是读过的某些书对自己产生了微妙的作用。

  • 当下的力量 / Eckhart Tolle 当意识到ego在自己身上的所作所为，开始对某些想法不再感到烦恼
  • 被讨厌的勇气 / 岸见一郎,古贺史健 "为遥远的将来设定一个目标，并认为现在是为其准备阶段。一直想着真正想做的是这样的事情，等时机到了才去做", 是一种拖延人生的生活方式。
  • 三体X：观想之宙 / 宝树 听说《三体》要拍成电视剧了，希望能重现当初读小说的震撼
  • 密室收藏家 / 大山诚一郎 今年读了不少推理小说，想起来可能是年初玩了一场剧本杀的缘故
  • 诡计博物馆 / 大山诚一郎
  • 绝对不在场证明 / 大山诚一郎
  • 消失的13级台阶 / 高野和明
  • 遗忘刑警 / 陈浩基
  • 醉步男 / 小林泰三 虽然不是恐怖小说，但读着读着就觉得毛骨悚然
  • 显微镜下的大明 / 马伯庸 亲王的文笔总是能够勾起对历史的遐想
  • 大明王朝1566 / 刘和平
  • 中共党史拐点中的人物与事件 / 李海文
  • 银河系边缘的小失常 / Etgar Kere 巧妙的构思让人回味无穷
  • 科幻世界译文版 2020全年合集
    • 虎狼之市 / Richard Morgan
    • 宇宙的边缘 / Terry Bisson 在浮躁的生活中能让人一读再读的文章真的是太难得了
    • 光点渐熄 / Kofi Nyameye
    • 白萩家餐桌风景 / 伴名练
    • 路魂 / 小川一水
    • 阿迪的选择 / 沈玫皙
    • 好彩 / Kim Stanley Robinson
  • 卓越密码：如何成为专家 / 田志刚 老板去年推荐的书目，抱着“希望成为像老板一样的专家”的心情读完了
