Booklist2019

發表於 2019-12-31 | 分類於 碎念

最近突然有种想法，即便是如今信息爆炸的时代，构造一个人精神世界的主要输入还是通过书籍。就个人体会，电影、剧集、音乐作品似乎都没有书籍来的深刻。今年算得上是精神财富和物质财富都丰收的一年，也亲身体会了一次从量变到质变的进步。物质方面不太方便展示，那就分享一下部分精神财富好了。😃

  • 漫长的告别 / Raymond Thornton Chandle 读过最文艺的悬疑小说
  • 恋情的终结 / Graham Greene 对作者把自己的嫉妒展现得毫无保留甚是敬佩
  • 自控力 / Kelly McGonigal 还是有点用的
  • 刻意练习 / Anders Ericsson 年初OffensiveCon上有个演讲专门就是以此书为指导修炼挖漏洞技术
  • 银河帝国系列 / Isaac Asimov 今年大部分的读书时间阿西莫夫的想象力所支配
    • 银河帝国4：基地前奏
    • 银河帝国5：迈向基地
    • 银河帝国6：基地边缘
    • 银河帝国7：基地与地球
    • 银河帝国8：我，机器人 系列最爱
    • 银河帝国9：钢穴
    • 银河帝国10：裸阳
    • 银河帝国11：曙光中的机器人
    • 银河帝国12：机器人与帝国
  • 永恒的终结 / Isaac Asimov 在学习过程中如果有两个选择，在时间允许的情况下，我总是倾向于更加困难的那个。
  • 神们自己 / Isaac Asimov 平行宇宙中的三人家庭真有意思
  • 牧羊少年的奇幻之旅 / Paulo Coelho 没有一段旅程是白过的，即使最后会回到出发的地方
