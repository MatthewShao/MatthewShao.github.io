Booklist2015-2016

發表於 2016-06-09

迷迷糊糊的大学生活就像坐上了一列不知道终点站的火车，我呆呆地望着窗外的风景，转瞬间，它就从清晰的轮廓退缩成为一个个点，当我再想极力从延伸的地平线寻找它当初存在的痕迹，却再也无法还原出它应有的样子。
悲伤的日期是会传染的，我相信今天的悲伤就是两年前的今天传染过来的。悲伤得染红了一大片存在或不存在过的点……当然也有一些闪闪发光的点，我现在要把它逐一捡拾起来，以后总有用到的时候。

  • SQL注入攻击与防御 / Justin Clarke 深入剖析SQL注入漏洞的成因和利用，全面详尽
  • 邓小平时代 / 傅高义 某课推荐读物，从外国人的角度看待时代巨人
  • 红太阳的升起 – 延安整风的来龙去脉/ 高华 同是某课推荐读物，原本没有想过会读得下去的，谁知道竟然我真的整本看完了。对草创时期的红色政权进行了相信是比较客观的剖析，从中能窥探到很多本质的问题。
  • 我只是敢和别人不一样 / 周宏翔 第一部分的主人公很多地方很像我呀，可是以后应该不会再破费看这类书了
  • 霍比特人 / J.R.R Tolkien 史诗级小说的开端
  • 魔戒三部曲 / J.R.R Tolkien 对虚构世界的风土人情的想象力最是惊人，但是整篇读完就觉得剧情有点平淡
  • Bayesian Methods for Hackers / Cameron Davidson-Pilon 实操性很强的一本书，但是因为对理论的缺失，读的时候只能连蒙带猜一知半解
  • 古董局中局 全集 / 马伯庸 开始读了就不愿意放下的那种小说。 很多细节都经得起推敲，有历史底蕴和出乎意料的情节
  • 侏罗纪公园 / Michael Crichton 童年挚爱电影，小说重温
  • 看海的人 / 小林泰三 看完想流泪的科幻短篇小说集
  • 亚特兰蒂斯： 基因战争 / A.G Riddle 有些情节感觉很牵强，已经没有读续作的欲望
  • 欺骗的艺术 / Kevin Mitnick 电话诈骗指南
  • 马伯庸作品集 / 马伯庸 三国配角演义，殷商舰队玛雅征服史等， 再次证明马亲王的脑洞之大，文笔之妙，研究之深
  • 龙与地下城 / 马伯庸 虽然有点短，却是十分有趣
  • 黄金时代 / 王小波 原来是这种东西， 嬉笑怒骂好不痛快
  • 金瓯缺 / 徐兴业 细节描写，历史场景还原，还有人物的心理斗争描写十分到位， 可见作者的功底深厚。
  • 海伯利安 四部曲 / Dan Simmons 个人推荐年度最佳书籍，宏大的世界观和特别的社会形态发展规律， 同样是不看才需要理由的吧。
